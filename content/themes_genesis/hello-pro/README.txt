INSTALL
1. Upload the hello-pro.zip folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Hello Pro theme.

For theme documentation, please visit https://thebrandid.com/hello-pro-theme-setup/.
