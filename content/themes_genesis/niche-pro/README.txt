NICHE PRO
https://niche.designbybloom.co/

INSTALL AUTOMATICALLY
1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. From there you'll see an option for “Upload” at the top of the screen. 3. Click the “Browse” button, navigate your computer files and select the .ZIP file for your theme.

INSTALL MANUALLY
1. Upload the Niche Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Niche Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure theme to your liking. To set up the theme like the demo, please visit https://docs.designbybloom.co.
