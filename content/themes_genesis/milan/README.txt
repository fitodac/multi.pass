MILAN PRO
http://my.studiopress.com/themes/milan/

INSTALL
1. Upload the Milan Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Milan Pro theme.

WIDGET AREAS
Featured Content - This is only visible on the blog homepage.
Sidebar - This is visible sitewide.
Footer 1 - This is left column in the footer widget area.
Footer 2 - This is middle column in the footer widget area.
Footer 3 - This is right column in the footer widget area.

CUSTOM BACKGROUND
This theme supports custom background, which is a native WordPress function.

ICON FONTS
The icons used in Milan Pro are courtesy of Font Awesome. http://fontawesome.io/

LOCALIZATION
The Milan Pro theme is translation ready.  More information about the translation process can be found here:http://codex.wordpress.org/Translating_WordPress/
