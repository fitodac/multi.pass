Easily manage "dead" blogs in your multisite install.

== Description ==

Provides a new section in the superadmin area under 'Sites' called 'Dead'.

Lists blogs, just like 'All Sites', but provides filters specific to finding
dead blogs.  You can filter by:

1. Last updated
1. Number of posts
1. Number of pages
1. If the blog still has the default theme

Also offers a bulk email feature where you can email all admins of a set of
blogs with some codes for personalization.

Note that you might need to be using a modern browser in order for all features
to work.

== Installation ==

Just like any other.

1. Upload the zip file to the '/wp-content/plugins/' directory
1. Unzip
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the new section in the superadmin area under 'Sites' called 'Dead'
